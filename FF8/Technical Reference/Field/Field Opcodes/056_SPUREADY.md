-   Opcode: **0x056**
-   Short name: **SPUREADY**
-   Long name: SPU Start

#### Argument

none

#### Stack

  
*0*

**SPUREADY**

#### Description

Sets the asynchronous timer to 0 frames. See [SPUSYNC](164_SPUSYNC) for details.

This does something to access hardware, but I don't know what. Calling it causes some drives to make noises.
