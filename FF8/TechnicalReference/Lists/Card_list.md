## Trade rules

| Value | Name       | Description                                                                                                                                             | Confidence |
|-------|------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|------------|
| 0     | NONE       | No cards are exchanged after the match.                                                                                                                 | High       |
| 1     | ONE        | The winner takes exactly one card of their choosing from the loser's deck (AI opponents pick by highest `power`; the human player picks interactively). | High       |
| 2     | DIFFERENCE | The number of cards exchanged equals a computed point differential based on final board-tile ownership.                                                 | Medium     |
| 3     | DIRECT     | Each card's final ownership is resolved individually via the board-result table, rather than by a card count.                                           | Medium     |
| 4     | ALL        | The winner takes the loser's entire 5-card deck.                                                                                                        | High       |

## Game rules

| Bit | Value      | Name               | Description                                                                                                                   | 
|-----|------------|--------------------|-------------------------------------------------------------------------------------------------------------------------------|
| 0   | 0x1        | OPEN               | Open rule — both hands are visible to both players.                                                                           | 
| 1   | 0x2        | SAME               | Same rule.                                                                                                                    | 
| 2   | 0x4        | PLUS               | Plus rule.                                                                                                                    | 
| 3   | 0x8        | RANDOM             | Random rule — hands are drawn randomly from the player's owned cards instead of chosen.                                       | 
| 4   | 0x10       | SUDDEN_DEATH       | Sudden Death — a drawn match immediately replays with the board reset.                                                        | 
| 5   | 0x20       | *(unresolved)*     | No value found yet                                                                                                            | 
| 6   | 0x40       | WALL               | Wall modifier — combines with `CARD_GAME_RULE_SAME` to display "Same Wall"; behavior with other rules unconfirmed.            | 
| 7   | 0x80       | ELEMENTAL          | Elemental rule — board tiles are randomly assigned an element.                                                                | 
| 29  | 0x20000000 | FORCE_GIVE_CARDS   | Meta flag — skips the match entirely and grants the dealt deck directly to the player's save (scripted/cutscene card grants). | 
| 30  | 0x40000000 | BOTH_AI_CONTROLLED | Meta flag — forces both participants to `PARTICIPANT_TYPE_NPC`.                                                               |
