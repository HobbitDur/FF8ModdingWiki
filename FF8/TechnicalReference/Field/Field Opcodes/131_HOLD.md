-   Opcode: **0x0131**
-   Short name: **HOLD**
-   Long name: Set PHS lock status

#### Argument

none

#### Stack

  
*Player-Character ID*

*Switchable?*

*Selectable?*

**HOLD**

#### Description

Sets whether the PC character can be moved/selected in the switch menu.
