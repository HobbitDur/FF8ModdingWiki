-   Opcode: **0x0E5**
-   Short name: **USE**
-   Long name: Use entity

#### Argument

none

#### Stack

none

#### Description

Opposite of [UNUSE](../01a-unuse/). Sets bit 1 (value 2) of this entity's flag word at offset +352, re-enabling the entity's processing/display.

PC handler: `SCRIPT_USE`.

## Function addresses

| Function | Address | Description |
|---|---|---|
| `SCRIPT_USE` | 0x51DD60 | Field script opcode handler (verified IDA function) |
