-   Opcode: **0x004**
-   Short name: **GJMP**
-   Long name: Global jump

#### Argument

*Script entry index*

#### Stack

none

#### Description

Global jump: pushes the return position (current instruction pointer + 1) onto the stack, then sets the instruction pointer to the entry point of the given script (index into the field's script entry-point table). Unlike [JMP](../002-jmp/), which is a relative jump inside the current script, GJMP can jump into any script of the field.

PC handler: `SCRIPT_GJMP` at 0x51C530.
