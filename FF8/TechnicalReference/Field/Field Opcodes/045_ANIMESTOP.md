-   Opcode: **0x045**
-   Short name: **ANIMESTOP**
-   Long name: Resume script, controlled animation

#### Argument

none

#### Stack

  
**ANIMESTOP**

#### Description

Returns this entity to its [base animation](02C_BASEANIME).
