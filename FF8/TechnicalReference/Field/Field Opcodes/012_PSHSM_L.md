-   Opcode: **0x012**
-   Short name: **PSHSM\_L**
-   Long name: Push signed from memory (long)

#### Argument

Memory adress.

#### Stack

... =&gt; ..., *value*

#### Description

Push the first four bytes (signed long) from **Argument** onto the stack.
