-   Opcode: **0x04C**
-   Short name: **WINCLOSE**
-   Long name: Close Message Window

#### Argument

none

  
*Message Channel*

**WINCLOSE**

#### Description

Close the last window created by [AMES](065_AMES). I haven't tried this for other types of windows...
