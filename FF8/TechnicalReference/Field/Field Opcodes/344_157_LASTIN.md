-   Opcode: **0x157**
-   Short name: **LASTIN**
-   Long name: Last dungeon in

#### Argument

none

#### Stack

  
*1*

**LASTIN**

#### Description

Enters the "sealed" last-dungeon (Ultimecia's Castle) state. Sets bit `0x0800` of the field `miscFlag`, then pops one value: if non-zero it locks **all** features by writing the full `map_seal` mask (items, magic, GF, draw, command, limit, resurrection and save all locked). It copies the seal to `MAP_SEAL` / `SG_SETTING.mapSeal`, rebuilds the reserve-member list (party members not currently in the active party are recorded in the var-map `unk777` table for later restoration), and calls `engine_restore_game_status`. The individual features are then re-enabled one at a time with [SEALEDOFF](../159-sealedoff/). See also [LASTOUT](../158-lastout/).

PC handler: `SCRIPT_LASTIN`.
