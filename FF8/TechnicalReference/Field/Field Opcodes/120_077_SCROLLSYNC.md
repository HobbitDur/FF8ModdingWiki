-   Opcode: **0x077**
-   Short name: **SCROLLSYNC**
-   Long name: Scroll Sync

#### Argument

none

#### Stack

  
**SCROLLSYNC**

#### Description

Pauses script execution until the last called SCROLL command finishes.
