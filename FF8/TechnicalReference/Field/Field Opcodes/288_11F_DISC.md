-   Opcode: **0x11F**
-   Short name: **DISC**
-   Long name: Prepare disc change

#### Argument

none

#### Stack

  
*Disc number*

**DISC**

#### Description

Sets up which disc is asked for when [DISCJUMP](../038-discjump/) is called. The handler pops one byte, stores it in a savemap field, and sets the global current-disc value to *Disc number* − 1 (0-based).

PC handler: `SCRIPT_DISC`.

## Function addresses

| Function | Address | Description |
|---|---|---|
| `SCRIPT_DISC` | 0x523390 | Field script opcode handler (verified IDA function) |
