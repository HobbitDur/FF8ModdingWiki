-   Opcode: **0x06E**
-   Short name: **KEYON**
-   Long name: Enable Key

#### Argument

none

#### Stack

  
*Key Flags*

**KEYON**

#### Description

Enables certain keys to be pressed after they are otherwise disabled (for example with [UCOFF](../../../Script/Opcodes/04E_UCOFF)
