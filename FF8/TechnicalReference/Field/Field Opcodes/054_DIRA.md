-   Opcode: **0x054**
-   Short name: **DIRA**
-   Long name: Direct to Actor

#### Argument

none

#### Stack

  
*Actor ID* (use PSHAC)

**DIRA**

#### Description

Make this entity rotate to face the given actor.
