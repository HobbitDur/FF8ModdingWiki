-   Opcode: **0x057**
-   Short name: **TALKON**
-   Long name: Talk script on

#### Argument

none

#### Stack

none

#### Description

Enables talking to this entity by clearing its talk-disabled flag (+587 = 0). No stack use.

PC handler: `SCRIPT_TALKON`.

## Function addresses

| Function | Address | Description |
|---|---|---|
| `SCRIPT_TALKON` | 0x51EBB0 | Field script opcode handler (verified IDA function) |
