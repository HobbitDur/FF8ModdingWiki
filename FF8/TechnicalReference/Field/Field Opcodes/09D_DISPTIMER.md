-   Opcode: **0x09D**
-   Short name: **DISPTIMER**
-   Long name: Display countdown timer

#### Argument

none

#### Stack

  
*X position*

*Y position*

**DISPTIMER**

#### Description

Display the countdown timer on the screen and start counting it down.
