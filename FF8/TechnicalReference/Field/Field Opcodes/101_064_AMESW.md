-   Opcode: **0x064**
-   Short name: **AMESW**
-   Long name: A?, Message, and Wait

#### Argument

none

#### Stack

  
*Message Channel*

*Field message ID*

*X position of window*

*Y position of window*

**AMESW**

#### Description

Pops four values (message channel, field message ID, window X, window Y) and pops up an auto-sized message window on that channel (width/height fitted to the text, then clamped to the screen), then pauses script execution until the player dismisses the window (returns 1 while open, 3 once closed). If the channel is already busy it retries (return 5). The X/Y give the window's top-left; unlike [MES](../047-mes/) no separate WINSIZE is needed.

PC handler: `SCRIPT_AMESW`.

## Function addresses

| Function | Address | Description |
|---|---|---|
| `SCRIPT_AMESW` | 0x529020 | Field script opcode handler (verified IDA function) |
