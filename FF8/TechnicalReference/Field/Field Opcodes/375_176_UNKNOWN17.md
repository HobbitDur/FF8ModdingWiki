-   Opcode: **0x176**
-   Short name: **UNKNOWN17**
-   Long name: (PC stub)

#### Argument

none

#### Stack

  
*Field map ID*

**UNKNOWN17**

#### Description

Pops the field map ID and calls a null function — **does nothing on PC**. On PSX this presumably prefetched data for the given field (it appears in field-loading contexts).

PC handler: `SCRIPT_UNKNOWN17` (renamed `SCRIPT_UNKNOWN17_nop_stub` in the community IDB).

## Function addresses

| Function | Address | Description |
|---|---|---|
| `SCRIPT_UNKNOWN17` | 0x5219F0 | Field script opcode handler (verified IDA function) |
