-   Opcode: **0x166**
-   Short name: **DISABLEANGELO**
-   Long name: Disable angelo

#### Argument

none

#### Stack

  
*0 or 1*

**DISABLEANGELO**

#### Description

It uses 1 at the Esthar concourse and the Ragnarok airlock. Uses 0 in the ragnarok cockpit.
