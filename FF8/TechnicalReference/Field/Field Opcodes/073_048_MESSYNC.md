-   Opcode: **0x048**
-   Short name: **MESSYNC**
-   Long name: Message Window Sync

#### Argument

none

#### Stack

  
*Message Channel*

**MESSYNC**

#### Description

After [AMES](../065-ames/) is called, the window it creates stays until either [WINCLOSE](../04c-winclose/).
