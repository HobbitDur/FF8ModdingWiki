-   Opcode: **0x074**
-   Short name: **DSCROLLA**
-   Long name: Scroll To Actor

#### Argument

none

#### Stack

  
*Actor Code (use PSHAC)*

**DSCROLLA**

#### Description

Scrolls the screen onto the given actor.
