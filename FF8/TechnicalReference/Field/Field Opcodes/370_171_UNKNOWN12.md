-   Opcode: **0x171**
-   Short name: **UNKNOWN12**
-   Long name: Wait for UNKNOWN11 turn

#### Argument

none

#### Stack

none

#### Description

Yields until the turn started by [UNKNOWN11](../170-unknown11/) completes (its sync opcode).

PC handler: `SCRIPT_UNKNOWN12` (renamed `SCRIPT_UNKNOWN12_turnsync` in the community IDB).

## Function addresses

| Function | Address | Description |
|---|---|---|
| `SCRIPT_UNKNOWN12` | 0x527AB0 | Field script opcode handler (verified IDA function) |
