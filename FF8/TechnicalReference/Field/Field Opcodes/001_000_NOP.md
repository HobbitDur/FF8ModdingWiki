-   Opcode: **0x00**
-   Short name: **NOP**
-   Long name: No operation

#### Argument

None.

#### Stack

No change.

#### Description

Performs no operation. The handler simply returns 0.

PC handler: `Script_NOP`.

## Function addresses

| Function | Address | Description |
|---|---|---|
| `Script_NOP` | 0x51C160 | Field script opcode handler (verified IDA function) |
