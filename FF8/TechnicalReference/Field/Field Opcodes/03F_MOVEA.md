-   Opcode: **0x03F**
-   Short name: **MOVEA**
-   Long name: Move to Actor

#### Argument

none

#### Stack

  
*Actor ID* (always use PSHAC)

*Distance to stop*

**MOVEA**

#### Description

Makes this entity move towards the given actor.
