-   Opcode: **0x181**
-   Short name: **UNKNOWN16**
-   Long name: Set Draw Point ID

#### Argument

none

#### Stack

  
*Draw point ID*

**UNKNOWN16**

#### Description

Assigns this draw point an ID. Draw points with identical IDs share Full/Drained status.
