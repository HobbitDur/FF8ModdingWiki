-   Opcode: **0x01**
-   Short name: **CAL**
-   Long name: Calculate

#### Argument

Operation to perform (see description).

#### Stack

..., *value1*, *value2* =&gt; ..., *result*

#### Description

Calculate *value1* **Argument** *value2* and push the *result* into the stack. Possible values for **Argument**:

`000 ADD` : Addition (was a link, target page never created)  
`001 SUB` : Subtraction (was a link, target page never created)  
`002 MUL` : Multiplication (was a link, target page never created)  
`003 DIV` : Division (was a link, target page never created)  
`004 MOD` : Modulo (was a link, target page never created)  
`005 MIN` : Minus (was a link, target page never created)  
`006 EQ` : Equal (was a link, target page never created)  
`007 GT` : Greater (was a link, target page never created)  
`008 GE` : Greater or Equal (was a link, target page never created)  
`009 LS` : Less (was a link, target page never created)  
`00A LE` : Less or Equal (was a link, target page never created)  
`00B NT` : Not Equal (was a link, target page never created)  
[`00C`` ``AND`](../00c-and/)` : And`  
[`00D`` ``OR`](../00d-or/)` : Or`  
[`00E`` ``EOR`](../00e-eor/)` : Exclusive Or`

Additional bitwise operations not used or named in the executable:

`00F NOT` : Not (was a link, target page never created)  
`010 RSH` : Right Shift (was a link, target page never created)  
`011 LSH` : Left Shift (was a link, target page never created)

PC handler: `Script_CAL` (evaluates the arithmetic/compare sub-operator; see AND/OR/EOR pages).
