-   Opcode: **0x00B**
-   Short name: **POPM\_B**
-   Long name: Pop to memory (byte)

#### Argument

Memory adress.

#### Stack

..., *value* =&gt; ...

#### Description

Pop *value* from stack and store the first byte in **Argument**.
