-   Opcode: **0x031**
-   Short name: **RANIME**
-   Long name: Resume script, animate

#### Argument

Model Animation ID

#### Stack

none

#### Description

Play an animation, then return the model to its base animation. The "R" (resume) variant of [ANIME](../02d-anime/): it starts the animation in play-once mode (`execution_flags` bit 0x40, entity+352) but returns 3 (done + yield) immediately without waiting for it to finish, so the script keeps running.

PC handler: `SCRIPT_RANIME` at 0x526890.
