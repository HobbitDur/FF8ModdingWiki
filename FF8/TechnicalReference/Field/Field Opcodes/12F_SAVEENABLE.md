-   Opcode: **0x12F**
-   Short name: **SAVEENABLE**
-   Long name: Set Save Enabled

#### Argument

none

#### Stack

  
*(0=disable, 1=enable)*

**SAVEENABLE**

#### Description

Enables or disables saving.
