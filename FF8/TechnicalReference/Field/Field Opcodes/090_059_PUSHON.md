-   Opcode: **0x059**
-   Short name: **PUSHON**
-   Long name: Push script on

#### Argument

none

#### Stack

none

#### Description

Enables "pushing" this entity by clearing its push-disabled flag (+585 = 0). An entity's "push" script is run when the player walks into it. Detection range is set with [PUSHRADIUS](../063-pushradius/). No stack use.

PC handler: `SCRIPT_PUSHON`.

## Function addresses

| Function | Address | Description |
|---|---|---|
| `SCRIPT_PUSHON` | 0x51EBF0 | Field script opcode handler (verified IDA function) |
