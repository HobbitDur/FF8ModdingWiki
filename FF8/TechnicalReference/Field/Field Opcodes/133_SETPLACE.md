-   Opcode: **0x0133**
-   Short name: **SETPLACE**
-   Long name: Set Area Display Name

#### Argument

none

#### Stack

  
*A number*

**SETPLACE**

#### Description

This controls what shows up at the bottom of the menu and in your saved game slots.
