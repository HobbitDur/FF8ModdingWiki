-   Opcode: **0x0B5**
-   Short name: **MUSICLOAD**
-   Long name: Load Music

#### Argument

none

#### Stack

  
*Music Track*

**MUSICLOAD**

#### Description

Preloads a new field music track. You can start the new track by calling [MUSICCHANGE](0B4_MUSICCHANGE).
