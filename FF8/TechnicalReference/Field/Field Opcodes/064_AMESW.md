-   Opcode: **0x064**
-   Short name: **AMESW**
-   Long name: A?, Message, and Wait

#### Argument

none

#### Stack

  
*Message Channel*

*Field message ID*

*X position of window*

*Y position of window*

**AMESW**

#### Description

Pop up a message window and pauses script execution until the player dismisses the window.
