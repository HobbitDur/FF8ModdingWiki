-   Opcode: **0x157**
-   Short name: **LASTIN**
-   Long name: Last dungeon in

#### Argument

none

#### Stack

  
*1*

**LASTIN**

#### Description

Disables features of the game pertaining to the last dungeon's mechanic (items, saving, etc)?
