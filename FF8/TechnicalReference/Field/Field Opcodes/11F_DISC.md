-   Opcode: **0x11F**
-   Short name: **DISC**
-   Long name: Prepare disc change

#### Argument

none

#### Stack

  
*Disc number*

**DISC**

#### Description

Sets up which disc is asked for when [DISCJUMP](038_DISCJUMP) is called.
