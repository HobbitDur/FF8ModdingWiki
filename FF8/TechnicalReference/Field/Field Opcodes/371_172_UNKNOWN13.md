-   Opcode: **0x172**
-   Short name: **UNKNOWN13**
-   Long name: Sound effect attribute check

#### Argument

none

#### Stack

  
*Sound channel?*

**UNKNOWN13**

#### Description

Pops one value (ignored on PC) and stores the result of the sound-effect attribute check (`SdEffectAttrCheck`) into temporary variable 0.

Used in Galbadian Missile Base when pushing the launcher into its bay, where it was supposed to control which channel the pushing sound plays through.

PC handler: `SCRIPT_UNKNOWN13` (renamed `SCRIPT_UNKNOWN13_se_attr_check` in the community IDB).

## Function addresses

|---|---|---|
| `SCRIPT_UNKNOWN13` | 0x520200 | Field script opcode handler (verified IDA function) |
