-   Opcode: **0x130**
-   Short name: **PHSENABLE**
-   Long name: Set PHS Enabled

#### Argument

none

#### Stack

  
*0=disable, 1=enable*

**PHSENABLE**

#### Description

Enables or disables switch submenu.
